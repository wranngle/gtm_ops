# Product Sense

This repo exists to make a specific capability legible:

> AI-first internal automation across voice agents, GTM workflows, revenue operations, and operator tooling.

## Positioning Rules

- Be honest that examples are synthetic.
- Do not imply Wranngle has paying voice-agent customers.
- Show the operating system behind the business, not a brochure for the business.
- Prefer working run commands, fixtures, tests, and screenshots over broad claims.

## Reviewer Questions To Answer Fast

- Can this person build internal tools for GTM/Ops/Finance?
- Can this person work with ElevenLabs agent workflows?
- Can this person use Python and SQL in a business automation context?
- Can this person design safe integration boundaries?
- Can this person make an agent-readable engineering environment?
